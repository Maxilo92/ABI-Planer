# Legacy-Skripte

Dieser Ordner enthält historische Einmal- und Migrationsskripte, die aktuell nicht über `package.json`-Scripts im regulären Betriebsablauf verwendet werden.

## Ziel

- Produktive Skripte in `scripts/` schlank halten
- Historische Tools weiterhin nachvollziehbar aufbewahren
- Versehentliche Ausführung im Tagesbetrieb vermeiden

## Hinweise

- Vor einer Nutzung immer zuerst den Einsatzzweck und die Datenwirkung prüfen.
- Für produktive Änderungen bevorzugt die aktiv gepflegten Skripte aus `package.json` verwenden.
- Sensible Datenoperationen nur mit dokumentierter Freigabe durchführen.
